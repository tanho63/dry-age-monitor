# rpi dry-age-monitor

Monitoring dry age fridge temperature and humidity. 

## hardware list

- Raspberry Pi Zero 2 WH (with microSD card, power supply etc)
- [BME690 sensor](https://www.pishop.ca/product/bme690-4-in-1-air-quality-breakout-gas-temperature-pressure-humidity/)
- [Adafruit Stemma/QT 5 port hub](https://www.adafruit.com/product/5625)
- [Adafruit Qwic -> female sockets cable](https://www.adafruit.com/product/4397)
- [Adafruit Qt to Qt cable, 400mm](https://www.adafruit.com/product/5385)

not yet used:
- [Adafruit EMC2101 fan controller](https://www.adafruit.com/product/4808) - for adding a fan control system
- computer case fan
